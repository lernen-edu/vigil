# Month 3 Labs

Four labs, done in order. Each has its own directory with a full spec. The lab attempt floor and the no-AI rule apply to every one. This is an AI-free zone month.

| Lab | Directory | Time budget | Floor | What you build |
| --- | --------- | ----------- | ----- | -------------- |
| 3.1 Subnetting Drills | `lab-01-subnetting-drills/` | 10 to 12 h | 90 min per drill set | Hand-solved subnetting to muscle memory; calculators refused |
| 3.2 Virtual Network Build | `lab-02-virtual-network-build/` | 14 to 16 h | 90 min | A two-subnet routed network and a documented routing table |
| 3.3 Protocol Captures | `lab-03-protocol-captures/` | 10 to 12 h | 90 min | Annotated DHCP, DNS, and TCP-handshake captures of your own traffic |
| 3.4 TryHackMe Networking | `lab-04-tryhackme-networking/` | 10 to 12 h | 45 min per stuck task | A consolidated network model and the no-flag habit |

Two labs carry scope rules you must honor exactly. Lab 3.2 and Lab 3.3 touch live network configuration and live traffic. You build and capture only on your own machines and your own lab network. Re-read each lab's scope section before you run anything.

Complete all four, commit a notebook entry for each, write the subnetting method sheet, and produce the end-of-month annotated network diagram (see `../deliverable.md`) before advancing to Month 4.
