# Lab Log

The tutor appends to this file as you complete labs. Each entry records the date, the lab, the month, and a one-line summary. The tutor reads this file every third Friday to select two cold-revisit targets. Do not edit it by hand.

## Format

```
YYYY-MM-DD | <lab-slug> | <month-NN> | <one-line summary of what you actually produced>
```
